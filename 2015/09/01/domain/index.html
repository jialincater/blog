域名转出记

发表于 2015-09-01   |   分类于 tech   |  

有朋友说域名在国内容易有可能被HOLD，加上自己的域名马上就要到期了，因此痛下决心要把域名转到国外的服务商那里去。

我的域名是在中国互联购买的，废了半天劲找回了密码然后登入账户写ticket申请域名转出，在下周一就得到了回复，要求我联系qq客服。

当然，域名要转出其实根本不用联系客服，只要你有域名授权密码就行。但是在中国互联的控制页面上死活也找不到密码。于是，只能继续联系客服。

于是我转战qq，客服人员确认了我的身份之后，冷冷的问了一句：“为什么要转出呢”，顿时问的我有点晕，我是这样回答的：“希望挂到国外的提供商去。因为我希望申请国外的研究生。国外解析会可靠一点”。然后客服没有再回答我，直接给我转了另一个客服。另一个客服给我发来了一堆东西：

域名转出： 域名所有者是个人的 ：
1.免责申明书 填好打印手写签字
2.转出表格填好 打印手写签字
3.身份证 正反面复印件 在上面签字
4.以上资料请邮寄至我司；

域名所有者是公司的：
1.免责申请书 填好打印盖公司公章
2.转出表格填好 打印 盖公司公章
3.联系人身份证 正反面复印件 在上面签字
4.公司营业执照副本 复印件 在上面盖公司公章
5.以上资料请邮寄至我司；

注：域名专员收到资料 审核没问题 会在7个工作日内给域名所有者邮箱发转出确认邮件 确认回复后 发转出密码 请注意查收邮件

所以，没什么可说的了，既然人家这么要求了，就按照人家的要求一步一步的做，折腾完这一套文书之后，我就顺风给那边公司寄了过去。过了一两天顺风提示我快件被签收，又过了一天，客服发邮件告诉我，要求我用域名管理邮箱发域名转出确认信。于是我就用admin@domain.name又发了一封确认信。翌日，中国互联就把域名的密码发给我了。

然后，我又跟我的域名接收方Europe Registry说我终于知道我域名的转出密码了，让他们赶快帮我转出。
ps1. 之所以选择这么个奇葩的域名接收方是因为我的域名比较小众，不像.com那样的那里都可以接受。
ps2. 域名转出貌似有规矩，刚注册几个月之内貌似不能转出，而且转出的同时必须续费。这个具体要咨询域名提供商。

很快，Europe Registry那边就回复了，说我的域名密码正确，转出已经开始初始化。两星期之后，我发现域名依然没有成功转出，然后又给Europe Registry发工单问问什么情况了，然后Europe Registry那边回复告诉我说我的密码还不对，让我给他们发正确的密码。。。然后我就急了，就把2周前那个成功初始化的邮件又给他发了回去。

Hi….
I am really confused about the status of the domain domain.name
In the mail 【[#UFW-893-78144]: Re: Incorrect Domain Password for domain.name】,you told me that【The auth code was correct, and the domain transfer has been initiated.】that happened in Aug 14th.
However, when I asked you about how is my transfer going yesterday, you answered me in this mail【[#XMR-810-35537]: Transfer about domain.name】that【Unfortunately the authorization code you have provided for the transfer of your domain is being rejected by the registry as incorrect.】
Please tell me what is wrong with that? I paste my passcode again here.
DOMAIN:
Password:
Target: Europe Registry

In addition I am sure this authorization code is correct.

Thanks
By Jialin

然后他回复跟我说，是他搞错了，实际上你的域名已经转出完成（我汗…），他会转告技术部门最后结束我域名的transfer. 过了一天，我接到邮件，说我域名转出完成。又过了一天，在Europe Registry的控制面板上就可以对域名进行各种操作了。

总的来讲，这次转出大概用了一个月的时间，来来回回写申请，寄东西，发工单还有最后Europe Registry那边长时间的没反应耽误了好多时间。能转出成功还是要说这两边的客服还不错。转出了而且续期了，顿时有一种神清气爽的感觉。
