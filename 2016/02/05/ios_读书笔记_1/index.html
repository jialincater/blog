iOS 读书笔记 - 1

Posted on 2016-02-05   |   In tech   |     |   visitors

第一章

The object that received the message is called target

接收消息的一方叫做target

The message sent is called the action

发出的消息叫做action

这章主要是手把手教着做了个app，没啥干货

第二章

Cocoa Touch Frameworks are collections of objective-C class

Cocoa Touch框架是oc的一组类

In Objective-C,we typically put an underscore at the begining of the instance varible name.

在oc中，一般在实例变量前面加一个下划线

The important difference between C structure and an Objective-C class is that a class has methods

c的结构和oc的类的主要不同点在于oc的类有方法

AClass Instance = [[AClass alloc] init];

分配内存并初始化

  • receiver: a pointer to the object being asked to excute a method
  • receiver:一个指向将要执行该方法的对象的指针
  • selector: the name of method to be excuted
  • selector: 方法的名字
  • argument: the values to be supplied as the parameters to the methods
  • argument: 方法的参数
Ainstance = nil;

最简单的释放内存的方法

a message sent to nil is ignored

一个向空指针发送的消息会被忽略

NSObject有三个方法 alloc、init和description

这三个方法可以被子类重写

for (NSString *item in items)

You can nit use it if you need to add or remove objects within the loop.

快枚举不能用于增删对象

Objective-C only allows single inheritance, so every class can only has one super class

oc只允许单继承，因此每个类只允许有一个父类

Each initializer begin with the word init

构造函数必须以init开头

designated initializer must initialize all the varibles

designated initializer必须初始化全部变量

instancetype can only be used in return type

instancetype只能用于返回类型

But in objective-C, you can not have two methods with the same selector and different return types

在oc，你不能有同名且参数或返回类型不同的方法

id is defined as a pointer to any object

id是一个指向任何对象的指针

isa pointer is a pointer to its class

isa指针是指向一个对象的类的指针，他描述了obj is a class
